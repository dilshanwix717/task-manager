# Task Tracker — Frontend

Next.js (App Router) client for the Task Tracker API: authentication, task management
with filtering + pagination, and real-time updates over Socket.IO.

See the [root README](../README.md) for full setup instructions and design decisions.

## Quick start

```bash
cp .env.example .env.local   # points the app at http://localhost:4000
npm install
npm run dev
```

Open `http://localhost:3000`. The backend must be running (see the root README).
